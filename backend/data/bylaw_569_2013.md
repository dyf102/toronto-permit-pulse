# 150.8 Garden Suites

## 150.8.10 General
### (1) Definition
A garden suite means an ancillary building containing one self-contained residential unit.

### (2) Requirement
A garden suite is permitted subject to compliance with the conditions set out in this Section.

## 150.8.20 Setbacks
### (1) Minimum Setback from Lane
If a garden suite abuts a lane, the minimum required setback from the lane is 1.5 metres.

### (2) Minimum Rear Yard Setback
The minimum rear yard setback for a garden suite is 1.5 metres, provided that it is not adjacent to a lane.

### (3) Separation Distance from Residential Building
The minimum separation distance between a garden suite and the primary residential building on the same lot is 5.0 metres.

## 150.8.40 Height and Area
### (1) Maximum Height
The maximum height of a garden suite varies based on the separation distance from the main building:
- If separation distance is less than 7.5m: maximum height is 4.0m.
- If separation distance is 7.5m or greater: maximum height is 6.0m.

### (2) Maximum Footprint
The maximum footprint of a garden suite cannot exceed the lesser of:
- 60 square metres, or
- 40% of the rear yard area.
